* Project Description
Craft112 is an open-world RPG sidescroller inspired by the popular game Terraria. In this game, you can explore the world, collect resources, fight enemies (which also fight back), and try your best to survive in this world.

* How to run the project:
python app.py

* Libraries needed:
noise, pygame

    Install via:
    pip install noise pygame

* Shortcuts
ctrl+q to force quit
escape to go back or pause
in the game, press c to spawn an enemy (if difficulty is not set to peaceful)
